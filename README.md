# API Rest Deno

 >  This is a REST API using Deno, Oak and PostgresSQL

 Be sure to install [https://www.postgressql.org/](postgresSQL) and edit the "config.ts"
 file with your own credentials

 

 ## RUN

  ```
 # This project uses Denon
  deno start
  ```

 # Routes


  ```
  GET       /api/v1/products
  GET       /api/v1/products/:id
  POST      /api/v1/products
  PUT       /api/v1/products/:id
  DELETE    /api/v1/products/:id
  ```
